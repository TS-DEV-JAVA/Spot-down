Installation

Download the correct version for your operating system from the Downloads page.

Extract the files to your preferred location and run the executable.

Usage

  1. Open Spot-Down.
  2. Log in to your Spotify account.
  3. Select tracks, albums, or playlists to download.
  4. Click "Download" to save them locally.
